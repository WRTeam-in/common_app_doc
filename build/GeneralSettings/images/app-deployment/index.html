General Requirements for All Screenshots​

  1. Resolution: Minimum 1280x720 pixels
  2. Format: PNG format with transparent background where applicable
  3. Content: Use latest version of Google Play Console
  4. Text: Ensure all text is clearly readable
  5. Highlighting: Key areas should be highlighted or annotated
  6. Personal Information: All sensitive information must be redacted
  7. Consistency: Maintain consistent zoom level and window size
  8. Language: All screenshots should be in English
  9. Theme: Use light theme for better visibility
  10. Quality: Use high DPI settings for crisp images

Tips for Taking Screenshots​

  1. Use a screen capture tool that can:

    • Capture specific areas
    • Add annotations
    • Export in high quality

  2. Before taking screenshots:

    • Set display to 100% scaling
    • Clear browser cache
    • Use incognito/private window
    • Maximize browser window

  3. After taking screenshots:

    • Crop to relevant content
    • Add highlights/annotations
    • Redact sensitive information
    • Save in PNG format

  4. Verify each screenshot:

    • Correct dimensions
    • Clear text
    • No sensitive data
    • Proper highlighting
